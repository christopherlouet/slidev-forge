# Installation

## Quick Install (recommended)

No installation needed — run directly with `npx`:

```bash
npx slidev-forge
```

## Global Install

```bash
npm install -g slidev-forge
slidev-forge --version
```

## From Source

```bash
git clone https://github.com/christopherlouet/slidev-forge.git
cd slidev-forge
npm install
npm run build
npm link
```

## Requirements

- **Node.js** >= 20
